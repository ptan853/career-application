# Manual Skill Evals

Run these before publishing the skill.

1. User gives only a JD link: agent researches target and creates a target profile.
2. Timeline is missing profile basics: agent stops and asks for missing fields.
3. Timeline has events but no target strategy: agent creates section strategy before selecting events.
4. Resume plan maps sections to events and asks for confirmation.
5. Event rewrite loop rewrites one event and waits for approval.
6. User asks for ATS resume: agent uses ATS-safe template and conventional headings.
7. User asks for designed resume: agent shows or offers style options and marks ATS risk.
8. User asks for final PDF before review: agent asks to confirm the draft first.
